# Streamlit Event Web Apps

A collection of interactive Streamlit web applications developed for youth-centered event booths and educational activities.

## Overview

This repository contains web applications created for booth activities at Seoul Youth Center events. The projects were designed to provide engaging and interactive experiences for participants through simple recommendation systems and personalized outputs.

## Projects

### 1. magic-mirror (Career Recommendation)
- Development Data: April 24, 2026
- Purpose: An interactive career recommendation application based on user choices and MBTI-inspired personality traits.

#### Features
- Category-based exploration (Humanities, Engineering, Arts)
- Personality-type calculation
- Career recommendations
- Storytelling-based user experience

### 2. coming-of-age keyword generator

A personalized keyword recommendation application developed for Coming-of-Age Day events.
- Development Date: May 11, 2026
- Purpose: A personalized keyword recommendation application developed for Coming-of-Age Day events.

#### Features
- User name input
- Adjective and noun selection
- Personalized encouragement message generation
- Image-based result display
